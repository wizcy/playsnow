# 魔塔H5版本 - 快速启动指南

## 🎮 立即开始

### 方法1: 使用启动脚本（推荐）

```bash
cd magictower-h5
./start_server.sh
```

然后在浏览器访问：**http://localhost:8000**

### 方法2: 手动启动

```bash
cd magictower-h5
python3 -m http.server 8000
```

### 方法3: 使用VS Code Live Server

1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## ⌨️ 游戏操作

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 | 对话继续 / 确认选择 |
| J | 楼层跳转（需要风之罗盘） |
| L | 查看怪物信息（需要圣光徽） |
| B | 保存游戏 |
| N | 加载游戏 |
| Q | 退出游戏 |

## 🐛 常见问题

### Q1: 游戏无法启动？
**A:** 确保通过HTTP服务器运行，不能直接双击打开HTML文件。

### Q2: 图片无法加载？
**A:** 检查 `resources/images` 目录是否包含所有图片文件。

### Q3: 地图显示异常？
**A:** 检查 `resources/levels` 目录是否包含0.lvl到27.lvl文件。

### Q4: 存档丢失？
**A:** 检查浏览器是否禁用了localStorage，或清除了浏览器数据。
